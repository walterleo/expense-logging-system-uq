# fronted

## Project setup
```
npm install
```

### Check .env.example and create your .env

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

### Run docs server
```
npm run doc:dev
```

### Build docs
```
npm run doc:build
```
